Energy Dashboard

- A web-application, made for Sprightful LLC, that allows its customers to sign in and see the benefits that Sprightful is bringing them in an aesthetically pleasing way. Within the application, customers are able to view their solar energy generated, grid energy consumed, and savings generated over a configurable length of time as well as control their thermostat setting.

- This application uses a technology stack of Python3 in coordination with the Django framework, PostgreSQL for data storage, Bootstrap from Twitter, jQuery, HTML, and CSS. Along with this, several API's are interfaced with from the app:

  • Neurio API - Energy info
  • Ecobee API - Thermostat
  • Stripe API - Billing

Visualizing Sorts

- A simple application I made using Java & JavaFX to practice implementing sorting algorithms while also creating a visual aid to further enhance understanding of each sort individually. User may select which sort they want to see as well as control the rate at which elements are being sorted.
